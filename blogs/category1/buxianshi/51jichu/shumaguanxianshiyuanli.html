# 数码管种类

工作方式：共阴和共阳

位数：一位和多位

# 显示各种数字的代码

具体工作原理不详细说了，感觉没必要记。

共阳：位选和段选同时为高电平亮

共阴：位选和段选同时为低电平亮

下面分别是0~9的两种数码管显示（段选）。

unsigned char code duan1[]={0x3f,0x06,0x5b,0x4f,0x66,0x6d,0x7d,0x07,0x7f,0x6f}; //共阴

unsigned char code duan2[]={0xc0,0xf9,0xa4,0xb0,0x99,0x92,0x82,0xf8,0x80,0x90}; //共阳

共阴位选端接地是开，共阳数码管接电源是开。

# 多位数码管的使用

# 段选和位选

段选:数码管一位有7段 或者8段（多一个小数点），选择每一段亮或者灭。

位选：选择多为数码管第几位亮。

# 显示原理

每次亮一位数码管，过极短的时间（1ms）亮下一个，这样肉眼就会看为所有数码管一直在亮。

# 消隐

消隐：在数码动态显示过程中，若进行片选切换时没有对上一片显示的内容进行清空，则会导致当前数码管中出现上一片内容的余影，从而使显示模糊，影响了整个显示效果。

清空就是让数码管全灭，再送下一个段选。 如P2=0x00或P2=0xFF。

这是在一位数码管连续显示不同段选，或者多位数码管中一类位选不变的数码管且连续显示多种段选中用到的。

在多位数码管中，在关闭位选的情况下先送段码，然后再送位选，延时后先关闭位选，再换段码，再开位选。这种情况不叫消隐，也不用消隐。
