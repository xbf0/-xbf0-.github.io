# 为什么要按键消抖

在实际按键操作过程中，按键之后按钮里面会有轻微抖动。

这种轻微抖动足以影响单片机的正常工作，常常会使单片机认为该按键被按了很多次。

以软件的方式对其进行消抖可以消除这项干扰。

# 怎么按键消抖

通常以软件的方式进行按键消抖。

通过编写程序来实现。

# 详细流程

首先判断该按键是否已经按下

如果是，延迟一小段时间

再次判断该按键是否按下

若果是，则判定为按键确实是被人按了

# 原理

按键之后所产生的机械抖动和人为按键都能使单片机认为这个键被按了，

要根据两者的不同来消除抖动。

两者的最大不同就是按键处于闭合状态时延续时间不同。

一般延迟5ms即可 ，5ms就是延迟一小段时间。

可以看看延迟函数<<（传送门）

# 程序

下面是一个以K1为按键的例子

if(K1==0)
{
  delay5ms();//有很多种延迟，这只是其中之一。
  if(K1==0)
  {
      //执行相应的程序。
  }
}
